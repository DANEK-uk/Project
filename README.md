# Учет расходов (Expense Tracker)

Простое консольное приложение для отслеживания ежедневных трат. Написано на Python

## Возможности

- Добавление расходов
- Фильтрация по дате и категории
- Подсчет общей суммы трат
- Сохранение в файл (опционально)

## Как запустить

```bash
python expenses.py
